## 💬 Roteiro de Entrevistas

**Pergunta 1:** O que é um join em SQL?  
**Resposta:** INNER JOIN → combina registros comuns; LEFT JOIN → mantém todos da tabela principal; RIGHT JOIN → mantém todos
da tabela secundária.  

**Pergunta 2:** Diferença entre regressão linear e logística?  
**Resposta:** Linear → prever valores contínuos (ex.: vendas); Logística → prever categorias (ex.: churn sim/não).  

**Pergunta 3:** Como você lida com dados faltantes?  
**Resposta:** Remoção, imputação (média/mediana), ou técnicas avançadas como KNN imputer.  

**Pergunta 4:** Cite um projeto de dados que você fez.  
**Resposta:** “Criei dashboards em Excel para relatórios gerenciais, e agora evoluí para Power BI com integração de Python.”  

**Pergunta 5:** Como você explica resultados para pessoas não técnicas?  
**Resposta:** “Uso analogias simples e foco no impacto no negócio. Ex.: churn é como clientes deixando uma loja.”
